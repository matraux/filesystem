# Introduction
This library is intended for handling with files.


# Installation and Requirements

The recommended way to is via Composer:
```shell
composer require matraux/filesystem
```

| file-system | compatible with PHP
|----|---
| 1.0.0 | PHP 8.3
| 1.0.9 | PHP 8.3
| 1.1.0 | PHP 8.4
